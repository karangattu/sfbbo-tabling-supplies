üé® Signage & Visual Branding
Digital Video/Photo Display

üé® Signage & Visual Branding
Enlarged Bay Area Project Map

üéÆ Interactive Displays & Games
Interactive Bird ID Game
 Instead of a single-user scope, use three large photos of shorebirds (e.g., Snowy Plover, Killdeer, Long-billed Curlew) so groups can identify field marks together. Lisa
üéÆ Interactive Displays & Games
Expanded Migration Map
 Enhance the existing migration map to include birds SFBBO researches and incorporate MOTUS data. Add "jaw dropping" fun facts that passersby can read. The display could be made more interactive with laminated maps and velcro to show complex migration routes. Jan, Lisa
üéÆ Interactive Displays & Games
Banding & MOTUS Display
 Create a unified display featuring: a shortened, modified mist net from the station's old stock, potentially with a fake bird in it; a string of different-sized bird bands or old/problem bands if possible; and a photo/map of MOTUS towers. Jan, Lisa, Debbie
üéÆ Interactive Displays & Games
Matching Games
 Create a game where participants match program pins to a mission poster. Another idea is a larger, "lift the flap" style game matching photos of birds to their unique nest types (e.g., cup, scrape, hanging). Debbie, Jan
